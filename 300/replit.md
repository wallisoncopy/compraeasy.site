# App Moldes de Roupas - Landing Page

## Overview

This is a Portuguese-language landing page for "App Moldes de Roupas" (Clothing Patterns App), designed as a marketing website to promote and sell digital clothing patterns. The site features a single-page design with a countdown timer, hero section with embedded video, and call-to-action buttons to drive conversions. The application targets the Brazilian market for sewing enthusiasts and fashion designers looking for ready-made clothing patterns.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Architecture
- **Static HTML/CSS/JavaScript**: Simple client-side only architecture using vanilla web technologies
- **Single Page Application**: All content is contained in one HTML file with sections for different content areas
- **Responsive Design**: CSS-based responsive layout optimized for mobile and desktop viewing
- **Component-based Styling**: Modular CSS classes for reusable UI components like buttons, containers, and timers

### Design Patterns
- **Progressive Enhancement**: Core content works without JavaScript, with enhanced features added via JS
- **Mobile-First Approach**: Responsive design prioritizing mobile user experience
- **Conversion Optimization**: Strategic placement of CTAs and urgency-creating elements like countdown timers

### User Interface Components
- **Countdown Timer**: JavaScript-driven timer creating urgency (5-minute countdown that resets)
- **Video Integration**: Embedded YouTube iframe for product demonstration
- **Smooth Scrolling**: JavaScript-enhanced navigation between page sections
- **Animation Effects**: CSS animations for buttons and interactive elements

### Content Strategy
- **Brazilian Portuguese Localization**: All content optimized for Brazilian market
- **Scarcity Marketing**: Countdown timer and limited-time offers to drive immediate action
- **Social Proof Integration**: Prepared structure for testimonials and conversion tracking

## External Dependencies

### Content Delivery Networks
- **Font Awesome 6.0.0**: Icon library from cdnjs.cloudflare.com for UI icons and visual elements

### Media Services
- **YouTube**: Embedded video content for product demonstrations (video ID: b-BCog35QoY)

### Analytics & Tracking
