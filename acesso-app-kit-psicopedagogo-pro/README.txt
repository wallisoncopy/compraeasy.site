Como publicar no GitHub Pages:

2) Settings -> Pages -> Branch main -> /root -> Save.
3) Acesse o link gerado.

Configuração:
- Edite no app.js:
  - CONFIG.kitLink (link único do kit)
  - CONFIG.whatsappLink (seu WhatsApp)
  - UPSELLS[].link (link de cada oferta)
